#### 排列(Permutation)

- N 个不同元素中取出 r 个不重复元素, 按顺序排列
  - P(n, r) = n! / (n-r)!
- P(n, r) = n \* P(n-1, r-1)
  - r1 有 n 中取法, 则剩下的极为 n-1 中取 r-1 中数据
- P(n, r) = r \* P(n-1, r-1) + P(n-1, r)

  - 取 n1 则有 r 中取法, 故共有 r \* P(n-1, r-1) 不取 n1 则有 P(n-1, r)中取法

- 圆排列 P(n, r) / r 2<=r<=n
- 项链排列 基于 圆排列 考虑翻转 P(n,r)/(2r) 3<=r<=n

- 全排列 元素的个数是可重复的
- 多重全排列 元素的个数是可重复的 且 个数是有限的

  - P(n;r1,r2,r3..rt) = n!/(r1! _ r2! _ ... rt!) = 8! / (2!2!2!) 'pingpang' 进行全排列
  - (a1 + a2 + ... at)^n = ∑(n!/(r1!..rt!))(a1^r1...at^rt) 二项式定理展开

- 相连与非相连 捆绑法 将不相连的 a 和 b 认为 1 个元素 计算总量减去捆绑的组合
- 分区域 隔板法 6 个区域由 5 个板划分

#### 组合(Combination)

- N 个不同元素中取出 r 个不重复元素, 组成一个子集
  - 因为不考虑顺序, 故 去除 r! 种排列 即 C(n, r) = P(n, r) / r! = n! / (r! \* (n-r)!)
- C(n, r) = C(n, n-r)
  - 左: 取出的组合数 右: 剩下的组合数
- C(n, l) _ C(l, r) = C(n, r) _ C(n-r, l-r)
  - 左: 先从 n 中选择第一批 l 个, 再从 l 中选择第二批 r 个
  - 右: 从 n 中中选择第二批 r 个, 再从 n-r 中选择第一批 l-r 个

* (a + b)^n = C(n, 0)a^n + C(n, 1) _ a^(n-1) _ b + C(n, 2) _ a^(n-2) _ b^2 ... + C(n, r) _ a^(n-r) _ b^r + C(n, r) \* b^n
* 格路模型
  - (0, 0) -> (m, n) 共有 C(m+n, m)种走法
    - 可以假设 共有 m+n 步中 选择 m 步 得到最终走法数量
* C(n, r) = C(n-1, r) + C(n-1, r-1)

  - (0, 0) -> (n-r, r) = (0, 0)->(n-r-1, r) + (0,0)->(n-r, r-1) = C(n-1,r) + C(n-1, r-r)

* 范德蒙德 恒等式
  - C(m+n, r) = C(m, 0)C(n, r) + C(m, 1)C(n,r-1) + ... + C(m, r)C(n, 0)
  - 左: 从 m 个 A 和 n 个 B 中取 r 个
  - 右: 从 A 中取 a 个 即需从 B 中取 r-a 个

- 无重组合

  - 模型: n 个不同的球放入到 r 个相同的盒子里 且每个盒子只能放一个球

- 可重组合 n 中不同分类的元素取出 r 个 C(n+r-1, r) C 上面带个杠

  - 模型: r 个相同的球 放入到 n 个有区别的盒子, 且每个盒子可为空或多于一个球

- 不相邻组合
  - 从 {1, 2, 3, ...n} 取 r 个不相邻的数进行组合, 即不存在相邻的两个数 j, j+1
  - C(n-r+1, r)

#### 总结

- 非重排列 P(n, r) n 个人中找 r 个人排队
- 非重组合 C(n, r) n 个球中取 r 个
- 可重排列 n^r n 个字母可重复选择的选取 r 个组成字符串 / 每个字符可重复排列
- 可重组合 C(n+r-1, r) n 种水果选 r 个 / 每个字符可重复的组合
- 多重全排列 n!/(r1!r2!). r1 个 a r2 个 b 拼接成一个 n 字符串 / n 的可重排列 除去 r1 和 r2 的可重排列

#### 算法

###### 字典序

- 尽可能长的公共前缀,变化限制在尽可能短的后缀

- a,b,c -> a,c,b -> b,a,c -> b,c,a -> c,a,b -> c,b,a

- ```javascript
  // * 数字数组 number[]
  const next_permutation = ans => {
    const len = ans.length;

    let a, b;
    let i = len - 1;

    while (i > 0) {
      if (ans[i - 1] < ans[i]) {
        a = i - 1;
        break;
      }
      i--;
    }
    if (i === 0) {
      // * 反转当前数组
      return ans.reverse();
    }

    i = a + 1;
    while (i < len) {
      if (ans[a] > ans[i]) {
        b = i - 1;
        break;
      }
      i++;
    }
    if (i === len) {
      // * 调换最后一个
      b = len - 1;
    }

    // * 交换值
    [ans[a], ans[b]] = [ans[b], ans[a]];

    let l = a + 1;
    let r = len - 1;
    while (l < r) {
      [ans[l], ans[r]] = [ans[r], ans[l]];
      l++;
      r--;
    }

    return ans;
  };
  ```

### 整数拆分

- 讲一个正整数拆分成若干个正整数之和
- 有序拆分(考虑顺序)
  - 3 1+2 2+1 二种
  - 自然数n 拆成r个自然数之和  C(n-1, r-1)
- 无需拆分(不考虑顺序)
  - 3 1 和 2 一种
