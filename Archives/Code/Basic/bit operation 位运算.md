#### 基础

```bash
&: 111 & 101 = 101
   101 & 001 =   1

|: 111 | 101 = 111
   101 | 001 = 101

^ : 111 ^ 101 = 010
    101 ^ 001 = 100

~: ～101 = 010

>>: 8 >> 1 = 8 / 2 = 4
<<: 8 << 1 = 8 * 2 = 16

```

#### 常用方法
- 乘除
```javascript
let a = 8
a << 1 // 8*2=16 => 8(00001000) => 16(00010000)
a >> 1 // 8/2=4  => 8(00001000) =>  4(00000100)
```

- 交换两数
```javascript
function swap(a, b) {
  a = a ^ b // a = a^b
  b = b ^ a // b = b^(a^b) = a^(b^b) = a
  a = a ^ b // a = (a^b)^a = b^(a^a) = b
}
```
