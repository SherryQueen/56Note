# 复杂度

| 排序算法  | 平均时间复杂度 | 最好时间复杂度 | 最坏时间复杂度 | 空间复杂度 | 稳定性(排序前后两个相等的数位置不变) |
| --------- | -------------- | -------------- | -------------- | ---------- | ------------------------------------ |
| Bubble    | O(n2)          | O(n)           | O(n2)          | O(1)       | 稳定                                 |
| Selection | O(n2)          | O(n)           | O(n2)          | O(1)       | 不稳定                               |
| Insertion | O(n2)          | O(n)           | O(n2)          | O(1)       | 稳定                                 |
| Quick     | O(nlogn)       | O(nlogn)       | O(n2)          | O(nlogn)   | 不稳定                               |
| Heap      | O(nlogn)       | O(nlogn)       | O(nlogn)       | O(1)       | 不稳定                               |
| Shell     | O(nlogn)       | O(n)           | O(ns) s 为步长 | O(1)       | 不稳定                               |
| Merge     | O(nlogn)       | O(nlogn)       | O(nlogn)       | O(n)       | 稳定                                 |
| Count     | O(n+k)         | O(n+k)         | O(n+k)         | O(n+k)     | 稳定                                 |
